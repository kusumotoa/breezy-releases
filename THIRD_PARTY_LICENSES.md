# Third Party Licenses

Breezy uses the following third-party libraries.

## Apple Frameworks

Breezy is built exclusively with Apple-provided frameworks and does not include
any third-party open-source dependencies.

The following Apple frameworks are used:

- **SwiftUI** — Declarative UI framework
- **SwiftData** — Persistent data storage framework
- **Foundation** — Core data types and utilities
- **CoreServices** — File system event monitoring (FSEvents)
- **AppKit** — macOS application framework

These frameworks are provided by Apple Inc. and are subject to the
[Apple SDK License Agreement](https://developer.apple.com/support/terms/).
